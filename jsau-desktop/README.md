jsau-desktop
============

Usage
-----

    $ npm install
    $ npm start

Alternatif : Lancement en fenêtre plein écran :

    $ npm start -- --max


Développements
--------------

Pour démarrage en mode debug, avec le panneau « devTools » ouvert :

    $ npm start -- --debug

ou

    $ node_modules/.bin/electron main.js --debug
